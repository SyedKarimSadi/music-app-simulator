# Music App Simulator (C Language)

This is a console-based Music App Simulator developed in C language using file handling.

## Features
- Admin Login
- Approve / Reject song requests
- View songs & pending songs
- Play & search songs
- Rate songs
- File handling (songs.txt, pending.txt, ratings.txt)

## Admin Credentials
- Username: admin
- Password: 1234

## Files Used
- songs.txt
- pending.txt
- ratings.txt

## How to Run
1. Compile the program:
   gcc music_app.c -o music_app
2. Run:
   ./music_app
